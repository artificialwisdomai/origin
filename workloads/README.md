# Workload Overview

A workload provides the following:

- A **service** is provided within a workload.
- A **ReST API** is provided for service lifecycle management.
- A **ReST API** is provided for user interaction.
- A **CLI** is provided to interact with the LCM and UI.
- A **Dependency list** is provided.
- A **README.md**.

## retroformer

The retroformer model uses a smaller language model and then retrieves responses
from a database. This occurs via gradient decent (ie. optimization) of both the
model and the database.

[ArXiv](https://arxiv.org/pdf/2112.04426)
